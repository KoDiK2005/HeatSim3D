# HeatSim3D
Практика для НИРа
-------------------
Программный комплекс для математического моделирования теплопередачи в сплошных 
средах. Реализует явную конечно-разностную схему FTCS для решения трёхмерного 
уравнения теплопроводности Фурье–Кирхгофа на равномерной декартовой сетке. 
Поддерживает независимые граничные условия Дирихле на каждой из 6 граней области, 
выбор материала из библиотеки твёрдых тел и визуализацию результатов в виде 
тепловых карт, изолиний и профилей температуры.
